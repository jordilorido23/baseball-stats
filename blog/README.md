# Front Office Analytics Blog

Data-driven insights for baseball front office decision-making.

## Directory Structure

- **posts/** - Published blog posts in Markdown format
- **figures/** - Generated visualizations and charts for blog posts
- **templates/** - Blog post templates and publishing tools

## Publishing Workflow

1. **Research & Analysis** - Run analysis in Jupyter notebooks
2. **Generate Figures** - Save publication-ready visualizations to `/figures`
3. **Write Post** - Use templates in `/templates` to draft posts in `/posts`
4. **Publish** - Export to Substack, Medium, or self-host

## Current Series

### 2025 Free Agent Analysis
- Focus on data-driven evaluation of 2025-26 free agent class
- Expected stats analysis, aging curves, contract valuations
- Target audience: Front office professionals, agents, serious analysts

## Style Guide

- **Tone**: Professional but accessible
- **Length**: 1500-2500 words per post
- **Data visualization**: Always include at least 2-3 charts
- **Actionable insights**: Every post should have clear takeaways
- **Citations**: Link to data sources (Baseball Savant, FanGraphs, etc.)
